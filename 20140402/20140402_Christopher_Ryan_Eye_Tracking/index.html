eye trackers


Zhang et al, 2010 (doi:10.1145/2168556.2168611)

  • useful for people with mobile disabilities
  • also used for product design and cognitive research
  • usually requires specialized equipment

review: Hansen & Ji, IEEE Trans Pattern Anal Mach Intell (2010). 

"Eye pupil localization with an ensemble of randomized trees"

Markuš et al, Pattern Recognition (2014).

  • Fast and uses only a webcam
  • Uses averaged result of 100 regression trees
  • Preliminary eye and face detection uses OpenCV

Clustering eye images by 

Pixel intensity comparisons

(Training the tracker using a big set of eye images)
. . .                     
At each node:
Do 10 times:
    1. Randomly choose 2 pixel locations.
    2. Compute the intensity difference between pixels in this same location for all images in the cluster.
    3.  Split the cluster into 2 subclusters containing images that are similar based on this feature.
Choose the statistically optimal splitting feature and recurse (i.e., split each new subcluster).

Regression trees for pupil tracking

Training, final step: Leaves contain clusters of similar eye images, so average the previously landmarked pupils for each one.
Predict: given a new eye image, use each tree to associate the new image with a leaf cluster and thus a pupil coordinate.

Fully trained ensemble used ~14000 annotated images of eyes (BioID face database), tree depth: 10, trees in ensemble: 100.
